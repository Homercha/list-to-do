# CONTRIBUTING
## Вимоги
Для роботи з проєктом необхідно:
* Visual Studio 2022;
* .NET SDK;
* Git.
## Клонування репозиторію
```bash
git clone https://github.com/username/task-manager.git
```
## Створення нової гілки
Перед внесенням змін створюйте окрему гілку:
```bash
git checkout -b feature/new-feature
```
## Оформлення комітів
Рекомендований формат:
```text
Add task deletion validation
Fix task numbering bug
Update README documentation
```
## Внесення змін
1. Створити нову гілку.
2. Виконати зміни.
3. Перевірити працездатність програми.
4. Створити коміт.
5. Надіслати зміни до GitHub.
## Pull Request
Перед створенням Pull Request необхідно:
* перевірити код;
* переконатися у відсутності помилок;
* оновити документацію за потреби.
## Стандарти коду
* використовувати зрозумілі назви змінних;
* дотримуватися форматування Visual Studio;
* додавати коментарі для складних ділянок коду.
